# Gravitee API Client

An API test client for Gravitee.io. Send requests, organize collections, and explore your APIs.

## Getting Started

```sh
pnpm install
pnpm run dev
```

The app runs at [http://localhost:3000](http://localhost:3000).

## Build

```sh
pnpm run build
pnpm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
